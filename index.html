A

    a

  • La etiqueta article se utilisa para especificar contenido independiente y autónomo.

B

    body
  • El elemento contiene todo el contenido de un documento HTML, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.
    • button
  • La etiqueta button define un botón en el que se puede hacer clic
    • br
  • La etiqueta inserta un solo salto de línea.

C

    class
  • Asigna una o más clases a un elemento, permitiendo aplicar estilos CSS o realizar manipulaciones con JavaScript
    • checked
  • Indica que un input de tipo radio o checkbox debe estar marcado por defecto.
    • content
  • Proporciona un valor asociado con el atributo name o http-equiv en un elemento "meta"

D

    div
  • La etiqueta div se utilisa como division o un seccion del documento html, el cual es ulizado como contenedor para elementos HTML - que luego se estiliza con CSS o se manipula con JavaScript.

F

    Form
  • La etiqueta se utiliza para crear un formulario HTML para la entrada del usuario. El elemento puede contener uno o varios elementos de un formulario
    • For
  • Especifica a qué elemento de formulario corresponde una etiqueta

H

    H1--H6
  • Las etiquetas h1--h6 se utilizan para definir encabezados HTML.
    • head
  • El elemento es un contenedor para metadata (datos sobre datos) y se coloca entre la etiqueta "html" y el "body" etiqueta
    • header
  • El elemento representa un contenedor para contenido introductorio o Un conjunto de enlaces de navegación.
    • HTML
  • a etiqueta representa la raíz de un documento HTML.
    • href
  • Especifica la URL de un enlace.
    • height

I

    Id
  • Define un identificador único para un elemento HTML.
    • img
  • La etiqueta img se utiliza para incrustar una imagen en una página HTML. Las imágenes no se insertan técnicamente en una página web; Imágenes están enlazados a páginas web. La etiqueta crea un espacio de retención para la imagen a la que se hace referencia.
    • input
  • Define o especifica un campo de entrada en el que el usuario puede introducir datos

L

    li
  • Define un elemento de lista. La etiqueta se utiliza dentro de listas ordenadas (ol) listas desordenadas (ul) y en Listas de menús (menú).
    • link
  • La etiqueta define el relación entre el documento actual y un documento externo recurso. se utiliza con mayor frecuencia para vincular a hojas de estilo externas o para agregar un favicon a su sitio web.
    • list
  • Asocia un campo de entrada con una lista de opciones predefinidas

M

    main
  • La etiqueta especifica el contenido principal de un documento. debe ser único para el documento. No debe contener ningún contenido que se repita en los documentos, como barras laterales, enlaces de navegación, información de derechos de autor, logotipos de sitios y formularios de búsqueda.
    • meta
  • La etiqueta define los metadatos sobre un Documento HTML. Los metadatos son datos (información) sobre datos. "meta" siempre van dentro del elemento "head", y se utilizan normalmente para especificar el conjunto de caracteres, la descripción de la página, palabras clave, autor del documento y configuración de la ventanilla.

N

    nav
  • La etiqueta define un conjunto de enlaces de navegación Los navegadores, como los lectores de pantalla para usuarios discapacitados, pueden usar este elemento para Determine si desea omitir la representación inicial de este contenido.
    • name
  • Especifica el nombre del elemento de formulario, utilizado para identificar los datos cuando se envía el formulario.
