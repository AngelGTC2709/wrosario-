Glosario

HTML

!

    !
  • ES una herramienta la cual se utilisa para crear la plantilla de html 5
    • !DOCTYPE
  • La declaración 1DOCTYPE no es una etiqueta HTML. Es una "información" para el navegador sobre qué tipo de documento esperar.

A

    a
  • La etiqueta a define un hipervínculo, que se utiliza para enlazar de una página a otra.
    • article
  • La etiqueta article se utilisa para especificar contenido independiente y autónomo.
    • alt
  • Proporciona un texto alternativo para una imagen, que se muestra si la imagen no puede cargarse.
    • acction
  • Especifica la URL a la que se enviará un formulario

B

    body
  • El elemento contiene todo el contenido de un documento HTML, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.
    • button
  • La etiqueta button define un botón en el que se puede hacer clic
    • br
  • La etiqueta inserta un solo salto de línea.

C

    class
  • Asigna una o más clases a un elemento, permitiendo aplicar estilos CSS o realizar manipulaciones con JavaScript
    • checked
  • Indica que un input de tipo radio o checkbox debe estar marcado por defecto.
    • content
  • Proporciona un valor asociado con el atributo name o http-equiv en un elemento "meta"

D

    div
  • La etiqueta div se utilisa como division o un seccion del documento html, el cual es ulizado como contenedor para elementos HTML - que luego se estiliza con CSS o se manipula con JavaScript.

F

    Form
  • La etiqueta se utiliza para crear un formulario HTML para la entrada del usuario. El elemento puede contener uno o varios elementos de un formulario
    • For
  • Especifica a qué elemento de formulario corresponde una etiqueta

H

    H1--H6
  • Las etiquetas h1--h6 se utilizan para definir encabezados HTML.
    • head
  • El elemento es un contenedor para metadata (datos sobre datos) y se coloca entre la etiqueta "html" y el "body" etiqueta
    • header
  • El elemento representa un contenedor para contenido introductorio o Un conjunto de enlaces de navegación.
    • HTML
  • a etiqueta representa la raíz de un documento HTML.
    • href
  • Especifica la URL de un enlace.
